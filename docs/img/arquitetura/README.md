# Imagens - Arquitetura

Esta pasta contém diagramas de arquitetura do sistema FoodTrack.

- `diagrama-arquitetura-geral.png` - Visão geral dos microserviços
- `diagrama-integracao.png` - Como os serviços se comunicam

- PNG ou SVG
- Resolução: mínimo 1920x1080
- Fundo transparente ou branco
