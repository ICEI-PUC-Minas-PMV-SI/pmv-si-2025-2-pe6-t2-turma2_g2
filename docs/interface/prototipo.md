# Protótipo - Frontend Web

## 1. Visão Geral

Este documento apresenta o protótipo de alta fidelidade da aplicação web do FoodTrack, desenvolvido com base nos wireframes e requisitos funcionais do sistema.

## 2. Ferramentas Utilizadas

- **Figma** para design de alta fidelidade
- Componentes responsivos e interativos

## 3. Telas do Protótipo

### 3.1. Login (T01)

![Protótipo Login](../img/web/prototipo-login.png)

**Funcionalidades:**
- Autenticação de usuários
- Validação de credenciais
- Recuperação de senha
- Interface responsiva

---

### 3.2. Dashboard - Mesas (T02)

![Protótipo Dashboard - Mesas](../img/web/prototipo-mesas.png)

**Funcionalidades:**
- Visualização do status de todas as mesas
- Indicadores visuais de ocupação
- Acesso rápido às comandas
- Criação de nova comanda

---

### 3.3. Comanda - Pedido (T03)

![Protótipo Comanda - Pedido](../img/web/prototipo-comanda.png)

**Funcionalidades:**
- Seleção de produtos por categoria
- Adição de itens ao carrinho
- Especificação de observações
- Envio do pedido para cozinha

---

### 3.4. KDS - Cozinha (T04)

![Protótipo KDS - Cozinha](../img/web/prototipo-kds.png)

**Funcionalidades:**
- Visualização em formato Kanban
- Gestão do fluxo de pedidos
- Atualização de status em tempo real
- Filtros por estação/categoria

---

### 3.5. Pagamento (T05)

![Protótipo Pagamento](../img/web/prototipo-pagamento.png)

**Funcionalidades:**
- Resumo detalhado da comanda
- Múltiplas formas de pagamento
- Opções de divisão de conta
- Aplicação de descontos
- Fechamento da comanda

---

### 3.6. Relatórios (T06)

![Protótipo Relatórios](../img/web/prototipo-relatorios.png)

**Funcionalidades:**
- Filtros personalizados por período
- Gráficos de vendas e performance
- Tabelas de dados detalhadas
- Exportação de relatórios

---

### 3.7. Usuários (T07)

![Protótipo Usuários](../img/web/prototipo-usuarios.png)

**Funcionalidades:**
- Listagem de usuários do sistema
- Cadastro de novos usuários
- Edição de perfis
- Gerenciamento de permissões
- Ativação/desativação de contas
