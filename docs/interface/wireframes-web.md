# Wireframes - Frontend Web

## 1. Visão Geral

Este documento apresenta os wireframes detalhados das telas da aplicação web do FoodTrack.

## 2. Ferramentas Utilizadas

- **Figma** para design de alta fidelidade
- **Draw.io** para fluxogramas

## 3. Telas Principais

### 3.1. Login (T01)
```
[Inserir wireframe: img/web/wireframe-login.png]
```

**Elementos:**
- Logo centralizada
- Campo e-mail
- Campo senha
- Botão "Entrar"
- Link "Esqueci minha senha"

---

### 3.2. Dashboard - Mesas (T02)
```
[Inserir wireframe: img/web/wireframe-mesas.png]
```

**Elementos:**
- Header com menu de navegação
- Grade de mesas (cards)
- Status visual (verde=livre, vermelho=ocupada)
- Botão flutuante "Nova Comanda"

---

### 3.3. Comanda - Pedido (T03)
```
[Inserir wireframe: img/web/wireframe-comanda.png]
```

**Elementos:**
- Barra lateral: categorias de produtos
- Grid de produtos com imagem e preço
- Carrinho de itens à direita
- Botão "Enviar para Cozinha"

---

### 3.4. KDS - Cozinha (T04)
```
[Inserir wireframe: img/web/wireframe-kds.png]
```

**Elementos:**
- Colunas Kanban: Pendentes | Em Preparo | Prontos
- Cards de pedidos com mesa e itens
- Botões de ação por card
- Filtro por estação

---

### 3.5. Pagamento (T05)
```
[Inserir wireframe: img/web/wireframe-pagamento.png]
```

**Elementos:**
- Resumo da comanda
- Seletor de forma de pagamento
- Opções de divisão
- Campo de desconto
- Botão "Fechar Comanda"

---

### 3.6. Relatórios (T06)
```
[Inserir wireframe: img/web/wireframe-relatorios.png]
```

**Elementos:**
- Filtros no topo
- Gráficos de vendas
- Tabela de dados
- Botão "Exportar"

---

### 3.7. Usuários (T07)
```
[Inserir wireframe: img/web/wireframe-usuarios.png]
```

**Elementos:**
- Lista de usuários
