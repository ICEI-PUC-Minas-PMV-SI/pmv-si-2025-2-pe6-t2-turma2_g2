# Wireframes - Frontend Web

## 1. Visão Geral

Este documento apresenta os wireframes detalhados das telas da aplicação web do FoodTrack.

## 2. Ferramentas Utilizadas

- **Figma** para design de alta fidelidade
- **Draw.io** para fluxogramas

## 3. Telas Principais

### 3.1. Login (T01)

![Wireframe Login](../img/web/wireframe-login.png)

**Elementos:**
- Logo centralizada
- Campo e-mail
- Campo senha
- Botão "Entrar"
- Link "Esqueci minha senha"

---

### 3.2. Dashboard - Mesas (T02)

![Wireframe Dashboard - Mesas](../img/web/wireframe-mesas.png)

**Elementos:**
- Header com menu de navegação
- Grade de mesas (cards)
- Status visual (verde=livre, vermelho=ocupada)
- Botão flutuante "Nova Comanda"

---

### 3.3. Comanda - Pedido (T03)

![Wireframe Comanda - Pedido](../img/web/wireframe-comanda.png)

**Elementos:**
- Barra lateral: categorias de produtos
- Grid de produtos com imagem e preço
- Carrinho de itens à direita
- Botão "Enviar para Cozinha"

---

### 3.4. KDS - Cozinha (T04)

![Wireframe KDS - Cozinha](../img/web/wireframe-kds.png)

**Elementos:**
- Colunas Kanban: Pendentes | Em Preparo | Prontos
- Cards de pedidos com mesa e itens
- Botões de ação por card
- Filtro por estação

---

### 3.5. Pagamento (T05)

![Wireframe Pagamento](../img/web/wireframe-pagamento.png)

**Elementos:**
- Resumo da comanda
- Seletor de forma de pagamento
- Opções de divisão
- Campo de desconto
- Botão "Fechar Comanda"

---

### 3.6. Relatórios (T06)

![Wireframe Relatórios](../img/web/wireframe-relatorios.png)

**Elementos:**
- Filtros no topo
- Gráficos de vendas
- Tabela de dados
- Botão "Exportar"

---

### 3.7. Usuários (T07)

![Wireframe Usuários](../img/web/wireframe-usuarios.png)

**Elementos:**
- Lista de usuários
- Botão "Adicionar"
- Formulário de criação/edição
- Status ativo/inativo

---

## 4. Fluxo de Navegação

```
Login → Dashboard → [Mesas | KDS | Pagamentos | Relatórios | Usuários]
```
